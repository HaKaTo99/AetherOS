# Vendor Outreach & Issue Templates

Use this template to open issues tracking vendor outreach for drivers and signing.

Title: Vendor partnership request — [Vendor Name] — [Device Family]

Body:
- Vendor: [Vendor Name]
- Contact: [email]
- Hardware: [device family]
- Requested: signed modules / firmware blobs / driver support / signing
- Proposed terms: [license, redistrib, support]
- Status: draft/outreach/signed

Assign to: `maintainer/team` and label `vendor` `driver` `partnership`.
