# Distribution Channels & Release Strategy

Options for distributing AetherOS to users and OEMs.

## Channels
- OEM preinstall (partner devices)
- Official download images (ISO / SD images)
- Cloud-hosted device images and edge deployment
- Package repository for updates (delta updates)

## Release Types
- Nightly (CI artifacts)
- Beta (QA tested)
- Stable (signed images and release notes)

## Update Mechanism
- Signed delta updates and rollback support
- OTA server for device fleets

## Marketplace
- Aether Store (optional) for apps and plugins with signing and review
